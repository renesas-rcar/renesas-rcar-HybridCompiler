HyCo supports https://github.com/onnx/onnx/blob/main/docs/Operators.md#Tan[onnx opset 7], https://pytorch.org/docs/stable/generated/torch.tan.html[torch.tan].

==== Attribute

None.

==== Restriction and Limitation

* CNNIP
** CNNIP does not support the `Tan` operator.

* DSP
** Supported input tensor dimension: [0, 7].
** Supported output tensor dimension: [0, 7].
